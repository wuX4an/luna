# Console

**Descripción:** Logging desde Lua en la consola del navegador.

### Funciones

```lua
console.log(...)
console.warn(...)
console.error(...)
console.info(...)
```

Ejemplo:

```lua
console.log("Hola desde Lua")
console.warn("Advertencia")
console.error("Error crítico")
console.info("Información irrelevante")
```
